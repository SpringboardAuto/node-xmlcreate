Class XmlText

Represents text in an XML document.

This text may only consist of character data, not markup. Restricted characters, such as the ampersand (&) and the opening angle bracket (<) are all automatically escaped.

To create an character reference or entity reference, you should use XmlCharRef or XmlEntityRef respectively instead.

XmlText nodes cannot have any children.
