    Static Public

    public DeclarationOptions: object source

    The options associated with the XML declaration. This object is used to create a new XmlDecl object.

    Properties:

    NameTypeAttributeDescription
    encoding string
    • optional

    The XML encoding to be included in the declaration. This value must be a valid encoding. If left undefined, no encoding is included.
    standalone string
    • optional

    The XML standalone attribute to be included. This value must be "yes" or "no". If left undefined, no standalone attribute is included.
    version string
    • optional

    The XML version to be included in the declaration. This value must be a valid XML version number. If left undefined, the default version is "1.0".

    public StringOptions: object source

    Formatting options for the string representation of an XmlNode and its children. This object is used by XmlNode#toString.

    Properties:

    NameTypeAttributeDescription
    doubleQuotes boolean
    • optional

    Whether double quotes or single quotes should be used in XML attributes. If left undefined, single quotes are used.
    indent string
    • optional

    The indent string used for pretty-printing. If left undefined, the default indent string is four spaces (" ").
    newline string
    • optional

    The newline string used for pretty-printing. If left undefined, the default newline string is "\n".
    pretty boolean
    • optional

    Whether pretty-printing is enabled. If left undefined, pretty-printing is enabled.
